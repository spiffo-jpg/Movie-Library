## Описание проекта

**Movie Library** — это графическое приложение на Python для ведения личной кинотеки. Программа позволяет хранить информацию о просмотренных фильмах, фильтровать их по жанру и году выпуска

### Основные возможности:

-  Добавление фильмов с полями: название, жанр, год, рейтинг
-  Отображение списка фильмов в таблице
-  Фильтрация по жанру (выпадающий список)
-  Фильтрация по году выпуска
-  Удаление выбранного фильма
-  Сохранение данных в JSON-файл
-  Загрузка данных при запуске
-  Валидация ввода (год — число, рейтинг 0-10)



## Установка и запуск

### Системные требования

- Python 3.7 или выше
- Tkinter 

### Пошаговая инструкция

```bash
# 1. Клонирование репозитория
git clone https://github.com/spiffo-jpg/movie-library
cd movie-library

# 2. Запуск приложения
python movie_library.py
